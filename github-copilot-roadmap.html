Agent Environments & Sessions

Choose where your agents run · Monitor & steer in real-time

💻

Local Agent

Runs in your IDE

  • • Full local tool access
  • • Uses your machine's hardware
  • • Best for: heavy lifting, debugging
⌨️

Copilot CLI

Background terminal agent

  • • Fleet mode for parallelism
  • • Creates PRs & issues from CLI
  • • Best for: automation, scripting
☁️

Cloud Agent

Runs on GitHub.com

  • • Isolated secure environment
  • • Persistent across sessions
  • • Best for: async issues → PRs

Agent Sessions: Monitor active sessions · Share & archive · Filter by status · Steer agents mid-execution · Full transparency into agent work

Copilot Plan Changes

CHANGED April 10–20, 2026

📋 What Changed

changelog →

  • • Paused new signups for Pro, Pro+, Student
  • • Copilot Free still open for new users
  • • Existing users can still upgrade
  • • Tighter usage limits on Pro tier
  • • Pro+ gets 5x+ the rate limits of Pro
  • • Refund option through May 20, 2026

🧠 Model Access Changes

  • • Opus models (4.7) → Pro+ only
  • • Opus 4.5 & 4.6 planned for removal from Pro+
  • • Pro trials paused for service quality
  • • Direction: basic tier (completions) + advanced tier (agents & premium models)
